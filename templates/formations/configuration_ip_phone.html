Formation Configuration IP Phone

Spécialisez-vous dans les systèmes de téléphonie IP et VoIP

Description de la Formation

Formation spécialisée en téléphonie IP pour maîtriser l'installation, la configuration et la maintenance des systèmes de communication VoIP en entreprise.

Objectifs Pédagogiques

  • Comprendre les protocoles VoIP (SIP, H.323, RTP)
  • Installer et configurer des PABX IP
  • Déployer des téléphones IP
  • Configurer la QoS pour la VoIP
  • Dépanner les systèmes téléphoniques

Programme Détaillé

Module 1 : Fondamentaux VoIP (20h)
  • Principes de la VoIP
  • Protocoles SIP et H.323
  • Codecs audio
  • Architecture VoIP
Module 2 : PABX IP (30h)
  • Installation Asterisk/FreePBX
  • Configuration des extensions
  • Routage des appels
  • Messagerie vocale
Module 3 : Téléphones IP (20h)
  • Configuration téléphones
  • Provisioning automatique
  • Fonctionnalités avancées
  • Softphones
Module 4 : Maintenance (10h)
  • Monitoring et supervision
  • Troubleshooting VoIP
  • Optimisation QoS
  • Sécurité VoIP

Équipements et Logiciels

  • Serveur Asterisk/FreePBX
  • Téléphones IP Cisco/Yealink
  • Casques et accessoires
  • Applications softphone
  • Outils de monitoring
  • Analyseurs de protocoles
